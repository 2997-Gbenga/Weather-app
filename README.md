## Project Description
[cite_start]I built a functional Weather Application that fetches and displays real live weather data from the free public Open-Meteo API[cite: 13, 102]. [cite_start]The application uses semantic HTML tags (including header, main, and section) to establish an accessible structural layout[cite: 72]. [cite_start]I designed a responsive user interface with CSS that automatically centers the app card layout on both desktop and mobile screens using media queries for narrow viewports[cite: 60, 77, 78]. [cite_start]The interface accurately matches the required design specifications, incorporating a dark navy title navigation bar (#1A3C5E), a teal/blue hero current weather display area (#2E86AB), and smooth transitions with hover effects on interactive elements[cite: 58, 59, 79, 81]. [cite_start]Using asynchronous JavaScript logic with the async/await pattern, the script handles data operations by translating city search queries into coordinates via a Geocoding API, capturing live local weather attributes (including humidity, wind speed, and UV index), converting WMO weather codes into custom icons, and displaying errors gracefully if a city is not found[cite: 69, 84, 85, 89, 92, 95, 96].
